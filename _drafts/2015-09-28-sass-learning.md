之前一直听说 [SASS](https://github.com/sass/sass) 和 LESS 等 CSS 编译工具，一直没有尝试，那天看了[阮一峰老师的文章](http://www.ruanyifeng.com/blog/2012/06/sass.html)，所以想要在我的项目里试一下 SASS。

## 安装 SASS

安装 SASS 非常简单，首先在 gem 中安装：

{% highlight %}
gem install sass
{% endhighlight %}
